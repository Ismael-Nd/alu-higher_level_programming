# python-if_else_loops_functions

Conditionals, loops, and functions in Python: `if`/`elif`/`else`, `for`
loops, and writing/importing functions.

## Tasks

| File | Description |
|------|-------------|
| `0-positive_or_negative.py` | Prints whether a random number is positive, zero, or negative |
| `1-last_digit.py` | Prints the last digit of a random number |
| `2-print_alphabet.py` | Prints the lowercase ASCII alphabet |
| `3-print_alphabt.py` | Prints the lowercase alphabet except `q` and `e` |
| `4-print_hexa.py` | Prints 0–98 in decimal and hexadecimal |
| `5-print_comb2.py` | Prints numbers 0–99 separated by `, ` |
| `6-print_comb3.py` | Prints all distinct combinations of two digits |
| `7-islower.py` | `islower(c)` — checks for a lowercase character |
| `8-uppercase.py` | `uppercase(str)` — prints a string in uppercase |
| `9-print_last_digit.py` | `print_last_digit(number)` — prints and returns the last digit |
| `10-add.py` | `add(a, b)` — returns `a + b` |
| `11-pow.py` | `pow(a, b)` — returns `a ** b` |
| `12-fizzbuzz.py` | `fizzbuzz()` — prints the classic FizzBuzz 1–100 |

## Environment

- Ubuntu 20.04 LTS
- Python 3.8.5
- pycodestyle 2.7.*
