# Python - Exceptions

Python functions handling and raising exceptions with `try` / `except` /
`finally` / `raise`, written for the ALU higher-level programming curriculum.

## Tasks

| File | Description |
| ---- | ----------- |
| `0-safe_print_list.py` | Prints x elements of a list, returns the number printed |
| `1-safe_print_integer.py` | Prints an integer, returns whether it printed |
| `2-safe_print_list_integers.py` | Prints the first x integers of a list |
| `3-safe_print_division.py` | Divides 2 integers and prints the result in `finally` |
| `4-list_division.py` | Divides 2 lists element by element |
| `5-raise_exception.py` | Raises a `TypeError` exception |
| `6-raise_exception_msg.py` | Raises a `NameError` with a message |
