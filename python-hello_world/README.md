# python-hello_world

Introduction to Python: printing, string manipulation, f-strings, slicing,
and running Python from shell scripts.

## Tasks

| File | Description |
|------|-------------|
| `0-run` | Shell script that runs a Python file named in `$PYFILE` |
| `1-run_inline` | Shell script that runs Python code stored in `$PYCODE` |
| `2-print.py` | Prints a string using `print` |
| `3-print_number.py` | Prints an integer with an f-string (no casting) |
| `4-print_float.py` | Prints a float with 2-digit precision |
| `5-print_string.py` | Prints a string 3 times and its first 9 characters |
| `6-concat.py` | Concatenates strings into a welcome message |
| `7-edges.py` | Slices a word into first 3, last 2, and middle parts |
| `8-concat_edges.py` | Builds a sentence from string slices |
| `9-easter_egg.py` | Prints "The Zen of Python" |

## Environment

- Ubuntu 20.04 LTS
- Python 3.8.5
- pycodestyle 2.7.*
