# Python - Data Structures: Lists, Tuples

Python functions manipulating lists, tuples and strings, written for the
ALU higher-level programming curriculum.

## Tasks

| File | Description |
| ---- | ----------- |
| `0-print_list_integer.py` | Prints all integers of a list, one per line |
| `1-element_at.py` | Retrieves an element from a list at a given index |
| `2-replace_in_list.py` | Replaces an element of a list at a specific position |
| `3-print_reversed_list_integer.py` | Prints all integers of a list in reverse order |
| `4-new_in_list.py` | Replaces an element in a copy of a list |
| `5-no_c.py` | Removes all `c` and `C` characters from a string |
| `6-print_matrix_integer.py` | Prints a matrix of integers |
| `7-add_tuple.py` | Adds the first two elements of 2 tuples |
| `8-multiple_returns.py` | Returns the length of a string and its first character |
| `9-max_integer.py` | Finds the biggest integer of a list without `max()` |
| `10-divisible_by_2.py` | Lists which integers are multiples of 2 |
| `11-delete_at.py` | Deletes the item at a specific position without `pop()` |
| `12-switch.py` | Switches the values of two variables |
