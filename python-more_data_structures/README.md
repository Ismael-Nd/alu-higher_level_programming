# Python - More Data Structures: Set, Dictionary

Python functions manipulating sets, dictionaries, lambda, map and more,
written for the ALU higher-level programming curriculum.

## Tasks

| File | Description |
| ---- | ----------- |
| `0-square_matrix_simple.py` | Computes the square of all integers of a matrix |
| `1-search_replace.py` | Replaces all occurrences of an element in a new list |
| `2-uniq_add.py` | Adds all unique integers in a list |
| `3-common_elements.py` | Returns the common elements of two sets |
| `4-only_diff_elements.py` | Returns elements present in only one of two sets |
| `5-number_keys.py` | Returns the number of keys in a dictionary |
| `6-print_sorted_dictionary.py` | Prints a dictionary by ordered keys |
| `7-update_dictionary.py` | Replaces or adds a key/value in a dictionary |
| `8-simple_delete.py` | Deletes a key in a dictionary |
| `9-multiply_by_2.py` | Returns a new dictionary with all values doubled |
| `10-best_score.py` | Returns the key with the biggest integer value |
| `11-multiply_list_map.py` | Multiplies list values by a number using `map` |
| `12-roman_to_int.py` | Converts a Roman numeral to an integer |
