# Python - Classes and Objects

This project introduces Object-Oriented Programming in Python through a
`Square` class, built up task by task.

## Concepts covered

- Defining classes and instantiating objects
- Private instance attributes and name mangling
- Attribute validation with `TypeError` and `ValueError`
- Getters and setters via the `property` decorator
- Public instance methods
- Module, class, and method documentation

## Files

| File | Description |
| --- | --- |
| `0-square.py` | Empty `Square` class |
| `1-square.py` | `Square` with a private `size` attribute |
| `2-square.py` | `Square` validating `size` at instantiation |
| `3-square.py` | `Square` with an `area()` method |
| `4-square.py` | `Square` with a `size` property and setter |
| `5-square.py` | `Square` with a `my_print()` method |
| `6-square.py` | `Square` with a `position` property |
