# python-import_modules

Importing modules, functions, and variables in Python, the `if __name__ ==
"__main__"` guard, and reading command-line arguments from `sys.argv`.

## Tasks

| File | Description |
|------|-------------|
| `0-add.py` | Imports `add` from `add_0.py` and prints `1 + 2 = 3` |
| `1-calculation.py` | Imports `add`, `sub`, `mul`, `div` from `calculator_1.py` |
| `2-args.py` | Prints the number and list of command-line arguments |
| `3-infinite_add.py` | Prints the sum of all command-line arguments |
| `4-hidden_discovery.py` | Prints the public names defined in `hidden_4.pyc` |
| `5-variable_load.py` | Imports the variable `a` from `variable_load_5.py` |

## Environment

- Ubuntu 20.04 LTS
- Python 3.8.5
- pycodestyle 2.7.*
